# CFO Command Center

An AI-powered finance intelligence platform for PE-backed companies.

Built by Manav Chouhan | June 2026

## What It Does

Two AI agents that automate the most manual work in a PE-backed CFO office:

**Variance Narrative Agent** — Upload actuals vs budget data. 
Get board-ready commentary generated by Claude automatically.
Replaces 3–4 days of manual variance commentary writing.

**Reconciliation Agent** — Upload a bank statement and GL export. 
Transactions matched automatically using three-layer logic. 
Exceptions flagged, categorized by risk, and explained by Claude.
Replaces 5–8 days of manual Excel matching every month-end close.

**Executive Overview** — Live summary of EBITDA performance, 
revenue variance, reconciliation match rate, and exception exposure 
in one CFO-ready dashboard.

## Business Context

Built as part of a study of AI transformation for PE-backed CFO offices.
Modeled on a fictional $55M cold storage logistics LBO (Project Irongate).
Illustrative business case: compresses financial close from 19 days to 5,
reduces finance G&A by $574K annually, creates $4.9M of incremental 
exit value at an 8.5x exit multiple.

Bank statement: date, description, amount, reference

GL export: date, description, amount, reference

## Run Locally

1. Clone this repository
2. Copy .env.example to .env and add your Anthropic API key
3. pip install -r requirements.txt
4. python -m streamlit run dashboard.py

The app works without an API key for all deterministic features.
An API key is needed only for Claude-generated commentary.
